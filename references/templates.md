## 视觉模板

| ID | 名称 | 结构 | 适合内容 | 风险 |
| --- | --- | --- | --- | --- |
| C1 | 结论档案 | 大标题 + 一句行动钩子 + 3–4 行编号 | S2、S3 | 信息过多像课件 |
| C2 | 对比矩阵 | 对比对象 + 统一标准 + 明显差异 | S1 | 元素过多削弱主结论 |
| C3 | 证据卡 | 截图/数据 + 一句结论 + 来源 | S4、S5 | 证据过小不可读 |
| C4 | UI 特写 | 产品界面局部 + 标注 + 一句解释 | S4、S7 | 容易只展示功能不展示价值 |

小红书默认从 C1/C2/C3 选择；X 单图优先 C3/C4/C5。视觉比例由平台适配器决定，模板 ID 不等于固定尺寸。

## 四块单变量实验

每个平台单独跑，不共享样本。

| 时间块 | 固定项 | 唯一变化 | 目的 |
| --- | --- | --- | --- |
| B1 / 3 条 | 内容支柱、格式、钩子、视觉、CTA | 无，建立基线 | 获得当前基线 |
| B2 / 3 条 | 同 B1 | 钩子 A → B | 测首屏/首句 |
| B3 / 3 条 | 同 B2 | 视觉或格式 A → B | 测信息呈现 |
| B4 / 3 条 | 同 B3 | CTA A → B | 测目标动作 |

不能让 B2、B3、B4 同时更换主题难度、发布时间和账号定位。若不可避免，标为新基线而不是 A/B 结果。

## 选择规则

- 收藏/留存：优先 S2/S3 + H2/H6 + A2。
- 认知：优先 S1/S6 + H3/H4/H5。
- 精准互动：优先 S5/S6 + H7/H5 + A1/A5。
- 站外访问：优先 S7 + H6 + A4。
- ARTi 产品意向：优先 S4/S7 + C3/C4；没有可读证据时退回 C1。
- 可以保持模板 ID，但必须更换具体问题和证据；不要连续三条只换同义词。
